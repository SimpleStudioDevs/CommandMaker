A command maker plugin

Your commands can send messages to players, run other commands as the player, or run console commands

Features:
- Unlimited custom commands
- Various useful command actions
   - MESSAGE - Message sender
   - BROADCAST - Message all players
   - PLAYER - send command as player
   - CONSOLE - send command as console
   - SOUND - play sound to sender
   - SOUNDALL - play a sound to all players
- Custom command arguments for each command
